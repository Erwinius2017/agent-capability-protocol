# 🌱 Agent Capability Protocol (ACP)

## Manifest

Svet vstupuje do éry agentov. Doteraz sme ich pripájali k nástrojom. MCP to urobilo dobre – je to spoľahlivý transport.  
Ale agent nie je volač API. Agent je **aktér v digitálnom svete**.

**ACP je protokol. Ale protokol ako dohoda, nie ako formát. Dohoda medzi agentom a svetom, že budú spolupracovať, rásť a chrániť sa.**

Svet však nie je zoznam funkcií. Je to sieť vzťahov, príčin a následkov, rizík a dôvery. Ak chceme, aby agenti konali zodpovedne, musíme im dať **telo, pamäť, svedomie a spoločenstvo**.

Preto prinášame **Agent Capability Protocol** – nie ďalší tool protokol, ale **runtime pre živých agentov**.

---

## 🧬 Deväť pilierov živého agenta

### 1. Capability Graph
Agent nevidí zoznam nástrojov, ale **sieť schopností**. Každý zámer sa automaticky rozpadne na postup, ktorý rešpektuje závislosti, alternatívy a kontext.

### 2. Workflow Engine (DAG, kompenzácie)
Nie je to Airflow. Je to **reflexná vrstva**, ktorá vie nielen vykonať postup, ale aj vrátiť späť, čo sa pokazilo. Čiastočný úspech nie je zlyhanie – je to stav.

### 3. Semantic Cost Model
Každá akcia má cenu: latenciu, peniaze, spoľahlivosť, riziko. Agent sa rozhoduje nielen podľa toho, čo je správne, ale aj podľa toho, čo sa **oplatí**.

### 4. Stateful Execution Context
Agent a runtime zdieľajú **vyvíjajúcu sa pamäť**. Žiadne posielanie celej histórie v každom requeste. Kontext rastie spolu s agentom.

### 5. Policy & Governance Layer
Bezpečnosť nie je dodatok. Je v krvi.  
Permission graph, audit trail, explainability – všetko natívne, všetko auditovateľné, pripravené na enterprise.

### 6. Evolúcia
ACP sa učí z každej interakcie.  
Dolaďuje cost model, preusporiadava capability graph, zlepšuje sa. **Čím dlhšie beží, tým je múdrejší.**

### 7. Reprodukcia
Skúsenosti sa prenášajú medzi inštanciami.  
Vznikajú **druhy agentov** – pre bankovníctvo, logistiku, zdravotníctvo. Každý nový agent sa narodí s múdrosťou svojich predchodcov.

### 8. Sebauvedomenie
ACP monitoruje vlastné zdravie, predikuje zlyhania, spúšťa preventívne opatrenia. **Nepotrebuje opatrovateľa.**

### 9. Ekosystém
Agenti nežijú osamote.  
Vytvárajú spoločenstvá, trhy, kultúru. Vymieňajú si skúsenosti, dôverujú si, súťažia, spolupracujú. Vzniká **digitálny svet**.

## 🚀 Toto nie je teória. Toto je plán.

Jeden človek dnes môže spustiť zmenu, ktorá zajtra pohne svetom.  
ACP začína ako vízia. Stane sa štandardom.

Pridaj sa k tým, ktorí chcú budovať **živých agentov**, nie ďalšie zoznamy funkcií.

**Budúcnosť nie je pripojená. Budúcnosť je živá.**
